## Debugging with VS Code

### Desktop app

Select the configuration `desktop (client)` in **Run and Debug**.

### Web app

```sh
cd client/webapp
npm run dev
```

Then select the configuration `webapp (client)` in **Run and Debug**.

## Testing Github actions

> [!IMPORTANT]
> Install [`act`](https://github.com/nektos/act).

```sh
act
```
